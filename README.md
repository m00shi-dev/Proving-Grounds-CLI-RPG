# Proving-Grounds-CLI-RPG
A simple CLI game to practice Python and git. The goal is to create basic game systems using python. Those systems include:

- Equipment management
- Item creation
  - Item mods
- Equipment management
  - Wearing gear
  - Stat calculations
- Enemy creation
- Basic combat
  - Damage calculation
  - Status effects by turn
  - Basic spells
- Inventoy management
- Leveling and XP System
  - XP for fighting enemies
  - Increased stats when level gained
